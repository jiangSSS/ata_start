1.打开命令窗口：在要提交的项目下右键--->Git Bash Here  2.从第一个命令开始： git status====查看状态，有修改过的就会红色显示 git add .====提交到临时仓库，注意后面有一个点 git commit -m '本次提交的备注'====提交并带上备注 git pull====拉去最新代码 git push====提交最新代码到仓库 3.git新建分支：git branch 分支名 4.第一次push代码到新建的分支：git push origin 分支名 5.切换到新建的分支：git checkout 分支名 6.登录gitlab---找到对应项目---刷新：可以看到Commit和Branch
